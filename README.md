# Task-List

Application using express and mongodb to update task list


### Installation- Docker Compose (Recommended)

1. Clone the repo

2. Install dependencies
    ```
    $ npm install
    $ npm start
    ```

3. Open localhost:3000


### Installation- Manually


1. Clone the repo

2. Install dependencies
    ```
    $ npm install
    $ npm start
    ```

3. Open localhost:3000
